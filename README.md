# Windows-Handle-Finder
一个能修改窗口的工具！
适用于 Windows 10-11 的可执行应用程序。
发布页面：[单击打开](http://www.yeskunkun.cn/news/?210.html)

这是一个轻量的小工具，能够使用`SetWindowLongPtrW`函数修改任何窗口的属性。

### 是什么
-----------
它能够编辑任何小于或等于其权限的进程的窗口。
其中，属性面板包括三大模块：
|模块|作用|
|----|-----|
|基本信息区|显示窗口的基本信息，单击即可复制。|
|操作窗口区|包括窗口的系统菜单和附加功能。|
|窗口属性区|可以添加、移除窗口属性。|
|窗口层级区|设置父窗口、生成测试窗口。|
|DWM属性区|用`DwmSetWindowAttribute`设置属性。|
|工具区|目前添加了权限工具（BypassUAC、SYSTEM）。|

>注意，这些功能只依赖`user32.dll`和`dwmapi.lib`。

### 为什么
啊，它能干的事情可多了！
属性区的“添加属性”能够使全屏的窗口 __强制窗口化__，或去掉最大化、最小化按钮等，方便调试窗口。

>移除了`WS_MAXIMIZEBOX`属性的窗口在强制最大化时会全屏，覆盖任务栏。

打开后，可以在示例窗口上按**Alt+X**组合键打开属性面板。

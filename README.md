### 是什么
-----------
它能够编辑任何小于或等于其权限的进程的窗口。
打开后，可以在示例窗口上按**Alt+X**组合键打开属性面板。
属性面板包括三大模块：
|模块|作用|
|----|-----|
|基本信息区|显示窗口的基本信息，单击即可复制。|
|操作窗口区|包括窗口的系统菜单和附加功能。|
|窗口属性区|可以添加、移除窗口属性。|
